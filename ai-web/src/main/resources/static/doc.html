🧠 规则引擎

🔴 红色预警 · 需立即关注

  • R1 销售额大幅下滑
    当天销售额环比昨日下降幅度 ≥ r1Threshold
  • R2 大量链接下跌
    当天下跌链接数 ≥ 当天总链接数 × r2Threshold%
  • R3 大量链接未出单
    当天未出单链接数 ≥ 当天总链接数 × r3Threshold%

🟡 黄色预警 · 值得注意

  • Y1 销售额小幅下滑
    下降幅度 ≥ y1Threshold 且 < r1Threshold
  • Y2 部分链接下跌
    未出单链接占比 < r2Threshold（与 R2 互斥）
  • Y3 部分链接未出单
    未出单链接占比 < r3Threshold（与 R3 互斥）

🟢 绿色提示 · 运营亮点

  • G1 销售额稳步增长
    当天销售额相较昨日增长幅度 ≥ g1Threshold
  • G2 上涨链接占比亮眼
    当天上涨链接数占当天总链接数比例 ≥ g2Threshold
